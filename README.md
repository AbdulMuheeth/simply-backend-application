# simply-backend-application


## Setup
  
### step-1 : clone this Repository
### step-2 : initialize the npm using cmd **npm init** ( be sure that all dependencies are installed )
### step-3 : Start server with cmd : **npm start**


## Routes
#### Login -> /login
#### PathObject -> /patchData
#### Thumbnail -> /createThumbNail
#### addAddress -> /addAddress
